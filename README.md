# 贴吧合并审核

## 食用方法
- 使用 tbMerge.init() 初始化 插件
- 使用 tbMerge.fastReply() 开启快速回复功能
- 使用 tbMerge.debug() 开启开发者模式(only console.log)
- 其他高级功能如UASwitch，anti-UEG,MarkPost,requestHistories请使用 **贴吧合并平台**
